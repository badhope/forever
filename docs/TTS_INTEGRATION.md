# TTS 语音克隆集成指南

## 推荐方案

### 1. Chatterbox TTS (推荐)

5秒零样本克隆，2025年最新开源SOTA，支持23种语言。

```bash
# 本地部署
git clone https://github.com/ResembleAI/chatterbox.git
cd chatterbox && pip install -e .
```

### 2. GLM-TTS

智谱开源，中文情感表达最强，支持零样本情绪克隆。

### 3. IndexTTS2.Pro

3-10秒样本，中文发音精准，支持多音字标注。

## 音频准备建议

✅ **推荐参数：**
- 时长：10-30秒清晰录音
- 采样率：24000 Hz
- 格式：WAV，单声道
- 环境：安静，无背景噪音

❌ 避免：
- 不要用有背景音乐的录音
- 避免多人同时说话
- 不要使用电话录音

## 伦理提醒

本项目声音克隆功能仅限用于：
1. 你本人拥有合法权利的声音
2. 逝者生前明确同意或无明确反对
3. 仅供私人怀念使用，不得公开传播

请尊重逝者，尊重声音权。
