# 头像与动态视频制作指南

## 静态头像

使用逝者生前清晰的正面照片即可。

**推荐工具：**
- **FaceSwapper** - 生成不同场景/表情的变体
- **CodeFormer** - 老照片修复、清晰度提升

## 动态说话视频

### SadTalker (推荐)

单张照片 + 音频 = 自然说话视频

```bash
git clone https://github.com/OpenTalker/SadTalker.git
```

支持：
- 自然唇形同步
- 头部微动、眨眼
- 表情迁移

### AniPortrait

腾讯开源，照片级真实感动画效果

### LivePortrait

更生动的表情驱动，支持微笑、皱眉等微表情

## 使用建议

1. 选择笑容温和的照片作为基准
2. 视频分辨率不用太高，720p足够
3. 不用追求完美的"逼真"，有那神韵就够了
4. 建议只在私人对话中使用，不对外分享
