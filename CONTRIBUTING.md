# Contributing to Forever

First off, thank you for considering contributing to this project. This is not
just another AI project. This is about memory, about love, about the people we
lost and will never forget.

Please contribute with care and respect.

## Code of Conduct

This project and everyone participating in it is governed by our Code of Conduct.
By participating, you are expected to uphold this code.

## What we need right now

1. **Better emotion models** - The PAD model can always be improved
2. **More personality tests** - Personality consistency validation
3. **Memory systems** - Better cognitive-inspired memory architectures
4. **Language support** - Currently Chinese only, help us internationalize
5. **Documentation** - Better explanations of the psychological foundations
6. **Voice samples** - Better TTS integration guides

## Development Setup

```bash
git clone https://github.com/badhope/forever.git
cd forever
npm install
```

## Pull Request Process

1. Ensure any install or build dependencies are removed before the end of the layer
   when doing a build.
2. Update the README.md with details of changes to the interface, this includes new
   environment variables, exposed ports, useful file locations and container parameters.
3. Increase the version numbers in any examples files and the README.md to the new
   version that this Pull Request would represent.
4. You may merge the Pull Request in once you have the sign-off of two other developers,
   or if you do not have permission to do that, you may request the second reviewer
   to merge it for you.

## Publishing and Attribution

- Do not use this technology for commercial purposes without serious consideration
  of the ethical implications.
- Credit the psychological foundations in all publications.
- If you build something on top of this, let us know. We'd love to see it.

And remember:

> Death is not the end. Forgetting is.

Build this project with love.
